Deployment Options

Choose the right approach for your security and operational needs

‚úÖ IMPLEMENTED

Option 1: Air-Gapped Deployment

Jenkins with zero internet access using pre-built custom image

Pros

  • Maximum security (zero attack surface)
  • Compliance-ready for regulated industries
  • Cost savings ($64/month vs Cloud NAT)
  • No ongoing internet charges

Cons

  • Manual plugin updates required
  • No automatic security patches
  • Initial image creation time (10-15 min)

Best For

  • Regulated industries (finance, healthcare)
  • High-security environments
  • Compliance requirements
  • Cost-sensitive deployments
Monthly Cost: $84.30

Option 2: Cloud NAT with Internet

Jenkins with outbound internet access via Cloud NAT

Pros

  • Automatic plugin updates
  • Automatic security patches
  • Easier maintenance
  • Standard deployment

Cons

  • Higher cost ($148.30/month)
  • Internet exposure (outbound)
  • Additional attack surface
  • Cloud NAT charges ($64/month)

Best For

  • Development environments
  • Less strict security requirements
  • Need for automatic updates
Monthly Cost: $148.30

Option 3: Temporary Public IP

Assign public IP temporarily for setup, then remove

Pros

  • Easy initial setup
  • Can install plugins during setup
  • Remove IP after configuration
  • Lower ongoing cost

Cons

  • Manual process to add/remove IP
  • Temporary internet exposure
  • Requires VM restart
  • Not fully automated

Best For

  • One-time setup scenarios
  • Hybrid approach
  • Testing environments
Monthly Cost: $84.30 (after removal)

Option 4: Bastion Host with Proxy

Use bastion host as proxy for Jenkins updates

Pros

  • Controlled internet access
  • Audit trail for connections
  • Can be automated
  • Maintains air-gap most of time

Cons

  • Additional bastion host cost ($25/month)
  • More complex architecture
  • Requires proxy configuration
  • Additional maintenance

Best For

  • Need occasional updates
  • Want audit trail
  • Hybrid security model
Monthly Cost: $109.30
